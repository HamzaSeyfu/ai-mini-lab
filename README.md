# AI Mini Lab

AI Mini Lab est un petit projet Python destiné à expérimenter une chaîne de travail Git automatisée.

Le projet simule un mini système de classification inspiré du machine learning.
Il prend des scores numériques en entrée, applique une logique de décision simple, puis retourne une catégorie.

Objectifs du projet :

- tester une structure propre de dépôt Git ;
- automatiser les commits et les pushs ;
- générérer automatiquement un post LinkedIn après chaque avancée ;
- servir de laboratoire léger pour des expérimentations IA.

## Exécution

```powershell
python .\src\main.py
```

## Exemple de sortie

```text
Input vector: [0.82, 0.15, 0.03]
Predicted class: Research-oriented AI profile
Confidence: 0.82
```
