# Technical Notes

Ce projet sert de bac à sable pour tester une chaîne d'automatisation personnelle.

La logique actuelle n'est pas encore un vrai modèle entraîné. Elle simule une décision de classification à partir de trois scores.

Évolutions possibles :

- ajouter un vrai dataset ;
- intégrer scikit-learn ;
- créer un modèle de classification simple ;
- ajouter des tests unitaires ;
- publier automatiquement les avancées GitHub avec un post LinkedIn généré.
